# python-foundations-lab
Python fundamentals lab with exercises, small scripts, tests, and notes as I build the base for applied AI projects.

## Current Focus

- Core Python syntax
- Functions and control flow
- Data structures
- File handling
- Testing basics

## Why This Exists

I’m using this repo to build public proof of consistent programming fundamentals.

## The Value of Each Exercise

1. Practices creating variables and displaying them with `print()` and f-strings.
2. Practices collecting user input and using that input in calculations.
3. Introduces conditional logic with `if`, `elif`, and `else`. Also practices normalizing user input with `.upper()`.
4. Introduces functions: how to define them and call them.
5. Reinforces conditional logic and user input handling.
6. Introduces lists, adding user input to a list, and looping through list items with a `for` loop.
7. Introduces dictionaries: storing related information with keys and values.
8. Combines lists and dictionaries to create a simple contact book.
9. Practices reading from a file and printing the file contents with Python.
10. Introduces `try` / `except` for handling user input errors without crashing the program.
